# Back-end Development - Capstone Project

This is the main orchestration repository for the Band Fan Club application, integrating the Pictures and Songs microservices into a unified Django frontend.

## Environment Setup
- **Created from template:** Yes
- **Successful execution of bin/setup.sh:** Yes
- **Python Version:** 3.9.x
- **Virtual Environment Name:** capstone-venv

## Features
- **Django Frontend:** User interface for fans
- **Microservice Integration:** Connects to external REST APIs
- **Containerized Deployment:** Dockerized and ready for Kubernetes/OpenShift
- **Database Migrations:** Automated schema setup for multiple databases
